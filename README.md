# MAG CRM - Modern CRM Frontend

A modern, production-ready CRM frontend built with Next.js 14, TypeScript, and Tailwind CSS.

**Live Demo:** [https://mag-frontend-cokx.onrender.com/](https://mag-frontend-cokx.onrender.com/)
**Backend:** [backend](https://mag-backend-0gn4.onrender.com/api/docs#/Roles/RolesController_list)

## Features

### Authentication & Security
- **JWT-based authentication** with secure session management
- **Role-based access control** (Admin, Manager, User, SuperAdmin)
- **Protected routes** with client and server-side guards
- **Token refresh** mechanism for seamless sessions
- **Responsive login/register** pages with demo account

### User Experience
- **Modern UI** with your brand colors (Primary: #e55a2b)
- **Fully responsive** design for all devices
- **Clean dashboard** with key metrics and activity feed
- **Notifications** and user profile management
- **Automotive-themed** interface for CRM

### Technical Features
- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Custom API client** with error handling
- **Session storage** for token management
- **React Hook Form** ready forms
- **Middleware** for route protection
